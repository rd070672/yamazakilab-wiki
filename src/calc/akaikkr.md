## 1. Akai-KKR でできる代表的な計算

- 電子状態（DOS・バンド構造に相当する情報）
- スピン分極状態の安定性
- 不規則合金の計算（CPA）
- 置換・希薄不純物系の DOS / 磁性解析
- グリーン関数法による局所状態密度（LDOS）
- スピン軌道相互作用・異方性評価（設定に応じて）

→ 特に周期性のない系・乱れの強い系に強い点が特徴。


## 2. 計算例その1：純金属（Fe）のスピン分極状態

### 目的
- bcc-Fe の強磁性解を得る
- DOS と磁気モーメントの確認

### 流れ（概要）
1. `in/fe` のような入力ファイルを用意
2. `./specx < in/fe > out_fe`
3. 出力から以下を確認：
   - 占有・フェルミ準位
   - スピン分極 DOS
   - モーメント値（サイト毎）

### 得られる典型的結果
- bcc-Fe で約 2.2 μB/atom 前後のモーメント
- majority / minority DOS が明確に分離

→ 基本動作確認として最適。


## 3. 計算例その2：不規則合金（CPA）Fe–Co 系

### 目的
- ランダム固溶体の平均電子状態をスーパーセルなしで計算
- 組成依存の磁気モーメントや DOS の傾向を調べる

### CPA の利点
- 架空の巨大スーパーセルを作らなくても  
  無限サイズのランダム系の平均特性が計算できる
- Fe–Co, Fe–Ni, Fe–Cr などに実績多数

### 入力の要点
- サイトごとに占有比率を設定：
- SITE 1 Fe 0.7 Co 0.3
- これを変えながら `x = 0.0 → 1.0` まで走らせる

### 出力で見るべき指標
- 組成依存モーメント（Slater-Pauling 曲線）
- DOS のスピン偏極
- フェルミ準位付近の状態密度

→ Fe–Co 系なら x ≈ 0.3–0.4 付近で最大モーメントが再現されることが多い。


## 4. 計算例その3：希薄不純物・置換欠陥

### 目的
- 低濃度ドーピングの局所電子状態を見る
- 例：Fe 母相中の Ir, Pt, B, C など

### 方法
- CPA で少量占有（例：1–5%）を設定
- 出力から確認するもの：
- 不純物サイト周りの LDOS
- モーメント変化
- スピン偏極率

### 応用例
- 軟磁性材料の微量添加効果
- Fe-Co-Ir のモーメント増加／異方性変化の機序検討
- ハーフメタル化の可能性調査


## 5. 計算例その4：スピングラス系への応用

### なぜ Akai-KKR で可能か？
- KKR-CPA は 乱れ（disorder）を平均場的に扱う理論
- スピングラスは
- 乱雑な交換相互作用
- ランダムなサイト占有
が本質的要素

→ 平均化した電子状態からスピングラス的傾向を評価できる。

### 典型的にできること
- 磁気モーメントの安定性判定（FM / AFM / NM）
- ランダム合金のスピン分極 DOS
- 元素置換に伴う局所モーメントの符号変化
- RKKY 的符号反転の示唆（間接交換の変調）

### 注意点（重要）
- Akai-KKR 単体ではスピングラス転移温度 $T_f$ は求まらない
- 実際には：
1. KKR-CPA で有効交換相互作用 $J_{ij}$ を評価
2. その $J_{ij}$ を用いて  
   古典スピン MC / mean-field / RKKY 解析 に渡す
- スピングラスは統計力学問題なので  
  KKR → 有効モデル → 別途シミュレーション という流れが現実的

### 関連する出力の見どころ
- 元素置換でモーメント符号が変わるか
- DOS でフェルミ準位直上・直下の対称性
- 磁気不安定性（近接に非磁性解が存在するか）

→ 例：Fe–Cr, Fe–Mn, Au–Fe などで典型的に議論。


## 6. 計算例その5：局所状態密度（LDOS）と表面・界面

- Akai-KKR はグリーン関数形式のため  
  面直交方向の層ごとの LDOS を扱う拡張が可能
- 例：
- Fe(001) 表面の表面状態
- 不純物が結晶欠陥近傍に与える影響
- スピン依存 LDOS の解析

→ 表面磁気異方性・表面磁化減少の定性的議論に応用できる。


## 7. 解析時のチェックポイント

- フェルミエネルギーの収束
- l_max・エネルギーメッシュの収束
- スピン収束（初期モーメント依存性あり）
- CPA の組成ステップの粒度
- 有効電荷・ポテンシャルの安定性

## 8. まとめ

- Akai-KKR は不規則合金・乱れ・稀薄不純物に強い第一原理コード
- 計算例として：
- 純金属のスピン分極
- CPA による組成依存解析
- 不純物効果
- スピングラス的傾向の検討（平均場的アプローチ）
- スピングラスそのものの熱力学は  
  KKR → 有効 $J_{ij}$ → 別手法という流れが必要
