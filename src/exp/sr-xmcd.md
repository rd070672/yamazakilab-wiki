## 概要

この「XMCD の基礎」では、

- XMCD がどのような現象を利用した手法か
- XAS との関係と吸収端の選び方
- 実験配置・検出モード・サムルールの概要
- 実験の流れと実務上の注意点
- 利点と限界

## 1. XMCD とは何か

XMCD（X-ray Magnetic Circular Dichroism: X線磁気円二色性）は、

- 磁化した物質に
- 右円偏光・左円偏光の X 線を入射したときの
- X 線吸収スペクトルの差

を観測する手法です。

簡単に言うと

- 磁化方向と同じ向きに「ねじれた」光（円偏光）と
- 逆向きに「ねじれた」光

が、異なる強さで吸収される現象が XMCD です。

その結果として

- 元素ごと
- 軌道（例えば 3d, 4f など）ごと

に、スピンモーメント・軌道モーメントに関する情報を取り出せるのが XMCD の特徴です。


## 2. XAS と XMCD の関係

XMCD は

- XAS（X-ray Absorption Spectroscopy）

の一種で、「偏光」を変えたときの差分を見ています。

- XAS（平均スペクトル）
  - 右円偏光と左円偏光の平均
  - 元素種、価数、配位環境などを反映
- XMCD（差分スペクトル）
  - 右円偏光スペクトル − 左円偏光スペクトル
  - スピン・軌道モーメント、磁気秩序を反映

したがって、解析では

1. まず XAS（平均）で吸収端のエネルギー位置やピーク形状を確認し、
2. XMCD（差分）から磁気情報を読み解く

という二段構えが基本になります。


## 3. どの吸収端を使うか（エネルギー領域）

典型的な例：

- 3d 遷移金属
  - 軟 X 線領域の L2,3 吸収端（2p → 3d 遷移）
  - 元素ごとの 3d バンドに直接感度があるため、磁性研究で最もよく使われる
- 希土類元素
  - M4,5 吸収端（3d → 4f 遷移）
  - 4f 電子のスピン・軌道モーメントに敏感
- 重元素（5d など）
  - L2,3 吸収端（硬 X 線領域）
  - スピン軌道相互作用が強い元素の XMCD に利用

どのエネルギーを使うかは

- 興味のある元素
- 見たい軌道（3d, 4f, 5d など）
- 利用可能なビームライン（軟 X 線か硬 X 線か）

によって決まります。


## 4. XMCD が生じる物理的な仕組み

円偏光 X 線は

- 光子一つ一つが角運動量（ヘリシティ）を持っている

と考えられます。

一方で、磁性体の中では

- スピンの配列（磁化方向）
- スピン軌道相互作用による軌道モーメント

によって、特定のスピン状態への遷移が起こりやすくなります。

すると

- 磁化方向と「同じねじれ」を持った円偏光と
- 逆向きのねじれを持った円偏光

では、許される遷移の確率が異なり、

- 吸収強度に差が生じる → XMCD シグナルとして観測される

というのが基本的なイメージです。


## 5. 実験配置とシグナル定義

実験では、概ね次の 2 つの方法で XMCD を取得します。

1. 光のヘリシティ（右円/左円）を反転させる
2. 磁場を反転させて、磁化方向を入射光に対して平行／反平行にする

理想的には

- 磁化方向 M
- 入射 X 線の波数ベクトル k

が平行（もしくは反平行）になるように配置し、

XMCD = μ⁺ − μ⁻

のように定義します（μ⁺, μ⁻ はそれぞれ右円・左円偏光で測定した吸収強度）。

実務的には

- ビームライン側でヘリシティ反転が可能かどうか
- 実験側で磁場反転がどれくらいの時間・安定度で行えるか

などを考えて、どちらを主に反転させるかを決めます。


## 6. 検出モード（TEY, TFY, 透過など）

XMCD は、XAS と同様に複数の検出モードで測定できます。

- TEY（Total Electron Yield）
  - 試料から放出される電子を検出
  - 情報深さが数 nm 程度で、表面・界面に敏感
  - 軟 X 線 L2,3 吸収端でよく用いられる

- TFY（Total Fluorescence Yield）
  - 蛍光 X 線を検出
  - よりバルクよりの情報（数十 nm ～ それ以上）
  - 自己吸収や飽和効果に注意が必要

- 透過（Transmission）
  - 薄膜や膜厚を揃えた粉末試料などで利用
  - 最もシンプルに吸収係数が得られるが、条件が限られる

同じ試料でも、検出モードによって見えている「深さ」や飽和の影響が変わるため

- 表面状態を見たいのか
- バルク平均を見たいのか

を意識してモードを選ぶことが重要です。


## 7. XMCD サムルールの概要

3d 遷移金属 L2,3 吸収端の XMCD では

- Thole, Carra らによるサムルール（sum rules）

を用いることで、

- 軌道モーメント $\m_orb$
- 有効スピンモーメント $\m_spin^eff$

を、スペクトルの積分から求めることができます。

おおざっぱなイメージとしては

- XMCD スペクトル（L3, L2）の面積
- XAS スペクトル（L3+L2）の面積
- ホール数（未占有状態の数）

などを組み合わせることで、比率として

- $\m_orb$ / $\m_spin$
- 元素ごとのモーメントの大きさ

を推定するという考え方です。

実際には

- 適切なバックグラウンド処理（ステップ関数・エッジジャンプの補正）
- 正しいホール数の見積もり
- 飽和効果・自己吸収の補正（必要に応じて）

など、慎重な前処理が必要になります。


## 8. 実験の流れと注意点（実務的な観点）

XMCD 測定の典型的な流れ：

1. 試料の準備
   - 励起したい吸収端に対して、適切な厚み・組成・表面状態にしておく。
   - 必要なら、その場でスパッタ・アニールなどの表面処理。

2. XAS の測定
   - まず偏光に依存しない XAS（平均）を測る。
   - エネルギースキャン範囲・ステップ・積算時間を調整。
   - 前縁（pre-edge）、後縁（post-edge）を含む範囲で測定しておくと正規化しやすい。

3. XMCD の測定
   - ヘリシティ／磁場を反転させながら、L2,3 吸収端をスキャン。
   - ドリフトを抑えるため、＋−＋− と交互に測るなどの工夫をする。
   - 単純な差分だけでなく、標準化した差（差分 / 平均など）もチェック。

4. 正規化と基礎解析
   - XAS の pre-edge, post-edge での正規化。
   - XAS のエッジジャンプを 1 に規格化してから XMCD を差分として扱う。
   - サムルールを使う場合は、積分範囲とバックグラウンド処理を慎重に設定。

注意点：

- 円偏光度（右・左の偏光純度）が 100% でない場合、その補正が必要。
  - SPring-8 BL25SUの場合、96%程度
- TEY では帯電（チャージアップ）に注意。
- TFY では自己吸収や再吸収による「ピークつぶれ」に注意。
- 強磁場を使う場合、ビーム位置の揺らぎや試料の機械的揺れにも注意。


## 9. XMCD の利点と限界

利点：

- 元素選択性
  - 例：Fe と Co のモーメントを分けて見る、など。
- 軌道・スピンモーメントの分離
  - サムルールにより、m_orb と m_spin^eff をある程度独立に評価できる。
- 薄膜・界面・多層膜への高い感度
  - 軟 X 線・TEY モードでは、実質的に表面数 nm の磁性を見ている。

限界・注意点：

- サムルールの適用には前提条件がある（スピン軌道分裂、電子数の仮定など）。
- 飽和・自己吸収・帯電など、測定系固有の影響が入りやすい。
- 複数元素・複数サイトが混在する系では、スペクトル解析が難しくなることも多い。
