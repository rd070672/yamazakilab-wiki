## 1. 放射光とはなにか

放射光（synchrotron radiation）は、

- 電子（あるいは陽電子）が
- 光速近くまで加速され、
- 磁場によって進行方向を曲げられたときに

その軌道の接線方向に放出される「非常に明るい電磁波」です。

### 主な特徴

- **高輝度（非常に明るい）**  
  通常の実験室 X 線源よりも桁違いに強い光子束を持つ。
- **高指向性（ビームが細い）**  
  強くコリメートされた光で、微小領域の測定に向く。
- **広いエネルギー領域**  
  VUV ～ 軟 X 線 ～ 硬 X 線まで、磁石や光学系の設計により調整可能。
- **偏光制御が可能**  
  直線偏光・円偏光（右／左）などを選べるため、XMCD などのスピン選択的測定に必須。
- **時間構造（パルス性）**  
  電子バンチに対応したパルス光であり、時間分解測定にも利用できる。

## 2. 放射光施設の構成（加速器と蓄積リング）

典型的な放射光施設は、以下の要素で構成されています。

1. **電子銃・線形加速器（LINAC）**  
   電子を生成し、数百 MeV 程度まで直線的に加速する。

2. **ブースターシンクロトロン**  
   LINAC から受け取った電子を、最終的な蓄積リングエネルギー（例：3 GeV, 6 GeV, 8 GeV）まで加速する。

3. **蓄積リング（storage ring）**  
   高エネルギー電子を周回させる本体。  
   電子がリングを周回する過程で、曲げ電磁石・挿入光源から放射光が放出される。

4. **挿入光源（insertion devices）**  
   - アンジュレータ（undulator）  
   - ウィグラー（wiggler）  
   周期的な磁場で電子の軌道を微小に振動させ、指向性の高い強い放射光を得る。

5. **ビームライン（beamline）**  
   蓄積リングから取り出した光を、
   - モノクロメータ（分光・単色化）
   - ミラー・レンズ（集光・整形）
   などで整えて、エンドステーション（測定装置）まで導く通路・光学系。

## 3. 放射光の物性研究への応用

放射光は、以下のような実験に広く使われます。

### 3.1 構造解析

- **X 線回折（XRD, SXRD, 粉末／単結晶回折）**
  - 高輝度・高角分解能を活かした結晶構造解析
  - 薄膜・界面・ナノ粒子の構造決定

- **小角散乱（SAXS / GISAXS）**
  - ナノスケール構造（ドメイン構造、ナノクラスターなど）の評価

### 3.2 電子状態・化学状態解析

- **X 線吸収分光（XAS, XAFS）**
  - 近傍構造（EXAFS）・価数・配位状態の解析
  - in-situ / operando 測定により、動作中材料の追跡も可能

- **X 線磁気円二色性（XMCD）**
  - 円偏光 X 線を用いたスピン・軌道モーメントの元素選択的測定
  - 軟 X 線 XMCD（3d 金属 L 吸収端）や硬 X 線 XMCD（重元素 L 吸収端）など

- **光電子分光（XPS, HAXPES, ARPES, SX-ARPES）**
  - 表面・バルクの電子状態・化学状態
  - フェルミ面・バンド構造の直接観測

### 3.3 イメージング・マイクロスコピー

- X 線顕微鏡 / ptychography / CDI など  
- 磁区構造観察（XMCD-PEEM, STXM）  
- 3D トモグラフィ（内部構造の 3 次元可視化）

## 4. 放射光実験の一般的な流れ

どの施設でも、放射光実験はおおむね以下のステップで進みます。

1. **課題申請**
   - 研究背景・目的
   - なぜ放射光でなければならないか（高輝度・元素選択性・時間分解など）
   - 必要なビームライン・エネルギー範囲・測定モード
   - 必要ビームタイム（シフト数）の見積もり

2. **実験準備**
   - 試料作製・前評価（XRD, VSM, XPS など）  
   - 試料形状・サイズ・マウント方法の検討
   - 真空・温度・磁場など、環境条件の確認
   - データ取得スクリプト・ログ整理方法の準備

3. **ビームタイム当日**
   - ビームアライメント・エネルギー調整
   - 測定条件（入射角、エネルギー、偏光、検出モード）の最適化
   - 本測定：スキャン計画に沿って、効率よくデータ取得
   - 簡易解析：その場でのチェック（異常の有無、再測定の要否）

4. **実験後の解析・報告**
   - データ整理・バックアップ
   - 解析スクリプト（フィッティング・スペクトル分解・構造モデルとの比較）
   - 施設への実験報告書（report）の作成
   - 論文・学会発表への展開

## 5. 放射光ビームラインの見取り図（概念）

典型的なビームラインは、ざっくり以下の構造をしています。

1. **光源部（ring からの取り出し口）**
   - アンジュレータ／ウィグラーなどの挿入光源

2. **一次光学系**
   - ミラー（集光・高次光除去）
   - モノクロメータ（分光・エネルギー選択）

3. **二次光学系・ビーム整形**
   - スリット（ビームサイズ・発散の調整）
   - 追加ミラー・レンズ（ナノビーム化など）

4. **試料室・エンドステーション**
   - 真空チャンバー
   - 試料ステージ（回転・傾斜・冷却／加熱）
   - 検出器（フォトダイオード、MCP、エネルギー分析器、2D 検出器など）

## 6. 安全と運用上の注意

放射光施設では、以下の点が重要です。

- **放射線安全**
  - 実験者は原則として、放射線教育を受講・資格登録が必要
  - ビームラインの遮蔽・インターロックを正しく理解すること

- **高真空・高電圧・磁場**
  - 真空装置の操作（ベント／排気手順）
  - 高電圧電源・マグネット電源の取り扱い
  - 試料持ち込み時のアウトガス・汚染に注意

- **限られたビームタイムの有効活用**
  - 事前シミュレーション・測定リストの準備
  - トラブルを想定した予備計画（PLAN B）を用意しておく
