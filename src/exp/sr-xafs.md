## 1. XAFS とは何か

XAFS（X-ray Absorption Fine Structure, X線吸収微細構造）は、

- ある特定元素の吸収端近傍で
- X 線吸収係数 μ(E) を精密に測定したときに現れる
- 微細な振動構造（Fine structure）

を総称したものです。

XAFS からは、主に次のような情報が得られます。

- 元素の価数・電子状態
- 近傍の原子種
- 最近接・次近接原子までの距離
- 配位数や熱振動・乱れの大きさ

つまり、元素選択的な「局所構造・局所電子状態」のプローブとして用いられます。

## 2. XANES と EXAFS

XAFS は便宜的に、エネルギー領域によって次の 2 つに分けられます。

1. XANES（X-ray Absorption Near Edge Structure）
   - 吸収端近傍（数十 eV 以内）の構造
   - 価数・電子状態・局所対称性などに敏感
   - スペクトル形状の比較・計算スペクトルとの照合で議論することが多い

2. EXAFS（Extended X-ray Absorption Fine Structure）
   - 吸収端を越えた高エネルギー側（数十〜数百 eV 以上）の振動構造
   - 吸収原子と周囲原子の干渉により生じる
   - 解析により、近傍原子の距離・配位数・動的・静的乱れを定量できる

実務的には、

- 吸収端付近の形状とエネルギー位置 → XANES
- 吸収端より高エネルギー側の振動部分 → EXAFS

として扱い、それぞれの情報を組み合わせて材料を理解します。

## 3. 測定の基本（透過・蛍光・電子）

XAFS は、吸収係数 μ(E) をエネルギーの関数として測る手法なので、実験的には

- 入射 X 線のエネルギーを連続的に掃引しながら
- 試料でどれだけ吸収されるかを測定する

ことが基本です。

代表的な検出モードは次の通りです。

1. 透過（Transmission）
   - 入射強度 I0 と透過強度 I を測り、μ(E) = ln(I0 / I) から求める
   - 均一なバルク試料で最もシンプルに扱える
   - 適切な厚み（エッジジャンプが 1 程度になる thickness）が必要

2. 蛍光（Fluorescence）
   - 試料から出る蛍光 X 線を検出
   - 希薄試料や厚い試料、基板上薄膜などで有利
   - 自己吸収・重なりピークなど、解析前に注意すべき点がある

3. 電子収量（Total Electron Yield など）
   - 表面敏感な情報を取りたい場合に用いる
   - 軟 X 線領域では特に有効

測定モードによって、得られる情報の「深さ」やアーチファクト（自己吸収、飽和など）が異なるため、試料形態に合わせたモード選択が重要です。

## 4. EXAFS の物理的イメージ

EXAFS 領域では、吸収原子から放出された光電子が、

- 周囲の原子で散乱されて戻ってくる経路
-直接外へ出ていく経路

の干渉によって、吸収係数に振動が生じます。

ざっくりしたイメージとしては、

- 吸収原子から半径方向に広がる電子波
- 近傍原子で散乱されて戻ってくる波
- これらの干渉が距離・配位数・散乱強度に応じて変化し、振動パターンを作る

というものです。

この振動成分をうまく取り出し、波数空間（k 空間）へ変換してフーリエ変換することで、r 空間（実空間）のピークとして近傍原子の情報を解析していきます。

## 5. 基本的な解析フロー（EXAFS）

EXAFS 解析の典型的な流れは次のようになります。

1. 生データ μ(E) の取得
   - I0, I, If などを使って μ(E) を計算
   - エネルギー軸のキャリブレーション（標準試料を用いる）

2. 吸収端位置 E0 の決定
   - 微分最大位置などから E0 を定める
   - E → k 変換（k は光電子の波数）

3. 背景除去・正規化
   - プレエッジのバックグラウンドを引く
   - ポストエッジを多項式でフィットし、エッジジャンプで正規化
   - 平滑な部分を除去し、振動成分 χ(k) を取り出す

4. k 空間での重みづけ
   - k^n χ(k)（n = 1, 2, 3 など）として高 k 領域を強調
   - データ領域を窓関数で切り出し（kmin, kmax）

5. フーリエ変換
   - χ(k) をフーリエ変換して r 空間のスペクトルに変換
   - おおよそのシェル構造（最近接、次近接など）がピークとして現れる

6. フィッティング
   - 理論計算（FEFF など）から得た散乱パスに基づき、
     - 配位数 N
     - 距離 R
     - デバイワラー因子 σ²（熱的・静的乱れ）
     などをフィッティングで求める

解析ソフトとしては、ATHENA（前処理）、ARTEMIS（フィッティング）、Larch などがよく使われます。

## 6. XANES の特徴と解析

XANES 領域では、EXAFS のような単純な波の干渉モデルだけでは不十分で、複雑な多重散乱が効いてきます。

ポイント：

- 吸収端位置
  - 価数の違いに応じてシフト（高い価数ほど高エネルギー側へずれることが多い）
- プレエッジピーク
  - 対称性の破れや特定の遷移（例：1s → 3d）の存在を反映
- メインピークとホワイトライン
  - 局所対称性・結合状態・バンド構造との対応

XANES 解析は、

- 既知の標準物質とのスペクトル比較
- 理論計算（多重散乱コード、DFT + 分光計算など）との比較

を通じて、「どのような局所構造・電子状態がこのスペクトルを生むか」を議論するスタイルが主流です。

## 7. 実験設計と注意点

XAFS 実験を行う際の実務的なポイント：

- エネルギーレンジとステップ
  - XANES 領域では細かいステップ（エッジ前後）
  - EXAFS 領域では一定の k ステップになるよう、エネルギーステップを設定
- 試料厚み
  - 透過モードでは、エッジジャンプが 1 前後になるように厚みを調整
  - 薄膜・粉末では基板やマトリクスの吸収も考慮する
- 標準試料
  - エネルギーキャリブレーション用の標準金属箔などを同時に測定
  - 既知構造をもつリファレンスとして XANES/XAFS を比較する

注意点：

- 自己吸収・再吸収
  - 濃い蛍光試料や高 Z 元素の蛍光測定ではスペクトルが歪む
- 放射線ダメージ
  - 高輝度ビームで長時間照射すると、試料が変質・損傷することがある
- ビームの安定性・エネルギードリフト
  - 測定中のドリフトが解析に影響するため、標準試料や参照チャンネルでモニタする

## 8. XAFS の利点と限界

利点：

- 元素選択性が高い
  - 混合系でも特定元素の局所構造を抽出できる
- 結晶性の良し悪しにあまり依存しない
  - アモルファスやナノ結晶・液体・高温高圧状態でも利用可能
- in-situ / operando 測定に適している
  - 電池、触媒、溶融金属など、動作中・反応中の状態を追跡可能

限界・課題：

- 解析にはモデルが必要
  - EXAFS フィッティングでは「どの元素がどの距離にいるか」という仮定を置く必要がある
- 複雑系ではパラメータが多くなりやすい
  - 相関の強いパラメータ（N と σ² など）をどう扱うかが難しい
- 蛍光・電子モードではアーチファクト（自己吸収、飽和、帯電など）が生じやすい
