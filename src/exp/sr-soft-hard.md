## 1. 軟X線・硬X線とは何か

### 1.1 エネルギーで見た分類

「軟X線」「硬X線」は、連続した X 線領域をエネルギー（波長）で便宜的に分けた呼び名です。

- 軟X線（soft X-ray）
  - おおよそ 100 eV ～ 数 keV 程度
  - 波長は 0.1 nm より長め（〜数 nm）
- 硬X線（hard X-ray）
  - おおよそ 数 keV ～ 数十 keV 程度
  - 波長は 0.1 nm 以下が典型的

境界は文献によって多少前後しますが、「侵入深さが浅く、表面・界面に敏感」な領域を軟X線、  
「侵入深さが深く、バルクに敏感」な領域を硬X線**とみなすのが実務的には分かりやすい整理です。

## 2. 物質中での挙動の違い

### 2.1 侵入深さ・吸収長

- 軟X線：
  - 物質による吸収が強く、侵入深さが浅い（数 nm ～ 数十 nm 程度が典型）。
  - 真空や低圧環境が必須で、多くの場合 UHV 実験（XMCD, ARPES, RIXS など）に用いられる。
- 硬X線：
  - 吸収が比較的弱く、侵入深さが深い（数 µm ～ mm オーダー）。
  - バルクの情報が得やすく、透過測定・粉末回折・トモグラフィなどに向く。

### 2.2 元素選択性と吸収端

- 軟X線：
  - 遷移金属 3d 軌道に対応する L 吸収端（数百 eV）、
  - 軽元素の K 吸収端（例：C, N, O の 200～600 eV）などがここに入る。
  - 価電子（3d, 2p など）と強く結合した吸収端が多く、電子状態・スピン状態に非常に敏感。

- 硬X線：
  - 重元素の K 吸収端（数 keV～数十 keV）や、3d遷移金属の K 吸収端（〜 5～10 keV）など。
  - 原子番号や局所構造（配位数・結合距離）に敏感で、EXAFS による構造解析に広く使われる。

## 3. 代表的な測定手法と特徴

### 3.1 軟X線でよく使われる手法

- **軟X線 XAS / XMCD**
  - 3d 遷移金属の L 吸収端での XAS, XMCD。
  - 元素選択的にスピン・軌道モーメントを評価できる。
  - TEY（電子収量）検出では表面数 nm の情報に敏感。

- **ARPES / SX-ARPES**
  - 角度分解光電子分光。
  - 軟X線（あるいは極端紫外線）でバンド構造・フェルミ面を測定。

- **RIXS（共鳴非弾性 X 線散乱）**
  - 共鳴励起状態からの散乱光を観測し、励起スペクトル（マグノン、フォノン、dd 遷移など）を調べる。

- **XMCD-PEEM / STXM などのイメージング**
  - 軟X線と顕微鏡技術を組み合わせ、磁区構造や化学分布を nm スケールで可視化。

### 3.2 硬X線でよく使われる手法

- **X線回折 (XRD) / 単結晶回折 / 粉末回折**
  - 結晶構造・格子定数・相分率を解析。
  - 高エネルギー X 線ではバルク深部まで透過可能で、in-situ や operando 測定に適する。

- **硬X線 XAFS（XANES / EXAFS）**
  - K 吸収端を用いた局所構造・価数解析。
  - 電池材料・触媒・高温高圧下の物質など広い分野で標準手法。

- **HAXPES（Hard X-ray Photoemission）**
  - 硬X線を用いた光電子分光。
  - 通常の XPS よりも情報深さが深く、埋もれた界面・バルク電子状態に敏感。

- **トモグラフィ・イメージング**
  - 吸収コントラスト・位相コントラストを利用した 3D 内部構造観察。

## 4. 実験条件・ビームライン設計の違い

### 4.1 光学系・真空条件

- 軟X線ビームライン：
  - グレーティングモノクロメータ・多層膜ミラーなど、波長が長めの光に適した光学素子を使用。
  - 空気中では強く吸収されるため、基本的に真空輸送。
  - サンプルも UHV チャンバー内に設置されることが多い（表面敏感なため、清浄表面が重要）。

- 硬X線ビームライン：
  - Si(111) などの結晶モノクロメータを用いた分光。
  - ビームライン全体を必ずしも UHV にする必要はなく、He ガスパスや空気中での取り扱いも可能な場合が多い。
  - 試料も大気圧・各種試料環境（セル、炉、電気化学セルなど）との両立がしやすい。

### 4.2 試料環境の自由度

- 軟X線：
  - 真空・低温・磁場環境など、物理量を精密に制御した基礎物性測定に向く。
  - 一方、大気中・液体・高圧環境などとの両立はやや難しく、専用セルが必要。

- 硬X線：
  - 透過性が高いため、セル越しに測定可能。
  - 電池実装体、触媒リアクター、金属溶融状態など、実稼働状態（operando）での測定に強い。

## 5. 材料研究での使い分け

「軟X線で見るか、硬X線で見るか」は、以下のような観点で選ぶことが多いです。

### 5.1 何を知りたいか

- 電子状態・スピン・軌道モーメントを元素選択的に知りたい
  - → 軟X線（特に L 吸収端）＋ XMCD / XAS / RIXS など。
- 局所構造・配位数・結合距離・価数を知りたい
  - → 硬X線 XAFS（K 吸収端）＋ EXAFS / XANES。
- 結晶構造・相転移・応力状態を知りたい
  - → 硬X線 XRD / 高エネルギー回折。
- 表面のバンド構造・トポロジー・表面電子状態
  - → 軟X線 ARPES / VUV-ARPES。
- 埋もれた界面やバルク電子状態
  - → HAXPES（硬X線光電子分光）。

### 5.2 試料形態・実験環境

- 薄膜・ヘテロ界面・表面敏感な現象：
  - 軟X線が有利だが、真空表面管理が重要。
- バルク試料・粉末・実デバイス：
  - 硬X線が有利で、セル越しの in-situ / operando が現実的。
