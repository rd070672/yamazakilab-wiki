# Wikiについて

--- 

- この Wiki は、tyamazakiの研究取り組みに関連する「計算・実験・データサイエンス」に関する知識とノウハウを整理し、主に研究室内外で共有・活用するための技術文書集です。
- 第一原理計算や分子動力学、マイクロ磁気解析、放射光実験、機械学習を用いたデータ解析など、日々の研究で実際に使っている手順・基礎・設定例・トラブルシュートを、wikiとして形に残すことを目指しています。
- Github pages × VitePress × VSCode でこのwebsiteを運用しており、編集や更新が容易なマークダウン形式で記述・更新しています。
- ChatGPT-5.1 thinkingの出力や関連資料を参考にwikiを作成しています。随時アップデートしていきます。

## 想定している読者

- 研究室内の学部生・大学院生  
- 共同研究者・関連分野の若手研究者  
- 材料科学・計算物理・データ駆動科学に関心のある学部3年生以上

数式やコードは必要に応じて扱いますが、できるだけ「なぜその設定にするのか」「どこでつまずきやすいか」を明示することを心がけています。

## コンテンツ構成

- 計算：第一原理計算、MD、フェーズフィールド、FEM/FDM、マイクロ磁気計算 など  
- 実験：材料合成、熱処理、磁気測定、低ノイズ計測、放射光実験のノウハウや測定条件・レシピ など 
- データサイエンス：Python による解析、機械学習、可視化、ワークフローの自動化 など
- 情報基盤：計算サーバの使い方、環境構築、スーパーコンピュータ など
- ラボ運営：研究の進め方やルール など

各ページは、目的に合わせて理解しやすいような内容や構成に調整しています。
参照先として可能な限り沢山の信用できる引用を付けます。

## 更新頻度

- 時間の取れるときにどんどん更新していきたいと思います。
- 手順書などの関しては、実際に動作確認がとれた方法や設定を掲載します。    
- 誤りや改善案があれば、ページ下部の “Edit this page on GitHub” から pull request や issue としてフィードバックいただければ幸いです。

## この Wiki から目指すこと

この Wiki を通じて、

- 「計算・実験・データ」を横断した再現性の高い研究基盤を整えること  
- 学部生・大学院生が、自分で環境構築から解析・考察まで完結できること  
- 研究室外の研究者とも、共通の言語で議論・共同研究できること  

を目標としています。

このサイトの内容が、皆さんの研究活動の少しでも役立てば幸いです。
